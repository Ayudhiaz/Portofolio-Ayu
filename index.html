HTML

Kemampuan dasar HTML meliputi pemahaman struktur halaman web, penggunaan tag dasar (seperti html, p, a, img) serta pembuatan form, tabel, dan pengaturan atribut seperti href, src, dan class.

CSS

Keahlian dasar CSS meliputi kemampuan memperindah tampilan halaman web dengan mengatur warna, font, layout, dan ukuran elemen. Ini termasuk penggunaan selector, properti seperti color, background, margin, padding, serta konsep dasar seperti box model, responsive design, dan penggunaan flexbox atau grid.

Ms Word

Keahlian dasar Word mencakup kemampuan mengetik dan mengedit dokumen, mengatur format teks (font, ukuran, warna), membuat daftar, menyisipkan gambar atau tabel, serta menggunakan fitur heading dan style untuk struktur dokumen. Termasuk juga kemampuan menyimpan dan mencetak dokumen dalam berbagai format.

Ms Excell

Kemampuan dasar Excel meliputi penggunaan lembar kerja (spreadsheet), input data, pemformatan sel, penggunaan rumus dasar seperti SUM, AVERAGE, dan IF, serta membuat tabel atau grafik sederhana untuk analisis data. Keahlian ini juga mencakup pemahaman dasar tentang sel, baris, kolom, dan range.

Ms Powerpoint

Keahlian dasar PowerPoint mencakup pembuatan presentasi menggunakan slide, memasukkan dan mengatur teks, gambar, grafik, serta animasi atau transisi antar slide. Pengguna juga mampu memilih tema desain, menyusun urutan slide, dan melakukan presentasi dengan baik.

Figma

Kemampuan dasar Figma mencakup pembuatan desain UI/UX secara kolaboratif. Ini termasuk membuat frame, menambahkan teks, bentuk, ikon, serta mengatur warna dan tata letak. Pengguna juga memahami dasar-dasar prototyping untuk menunjukkan alur navigasi antar halaman dan mampu menggunakan komponen serta auto-layout.
